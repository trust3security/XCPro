# Sources and version notes

## Primary reference: SC3 Annex A 2025

- Document: FAI Sporting Code Section 3 - Annex A (Gliding Championships), 2025 edition
- URL: https://www.fai.org/sites/default/files/sc3a_2025.pdf

Key sections used in this spec:

- 6.1-6.3 - task types and Racing Task definition, including the recommended >=45 degree leg angle
- 7.4 - starting, including line start, PEV option, cylinder start, and tolerance handling
- 7.6 - turn points and assigned areas, including the 500 m RT turnpoint cylinder
- 7.8 - finishing, including finish ring vs finish line and finish direction
- 7.9 - task completion definition

## Historical compatibility reference: SC3 Annex A 2013

- Document: FAI SC3 Annex A 2013_1
- URL: https://www.fai.org/sites/default/files/documents/sc3a_2013_1.pdf

Why keep it:

- older task definitions and local procedures may still use concepts like a Start Ring
- the 2013 text is explicit about legacy ring-style starts

Use it only for compatibility modes and older import/export support.

## Context reference: Sporting Code Section 3 2025

- Document: FAI Sporting Code Section 3 - Gliding, 2025 edition
- URL: https://www.fai.org/sites/default/files/sc3_2025.pdf

Annex A is the competition-rules layer. Section 3 is broader and is useful for
context, definitions, and current document status.
