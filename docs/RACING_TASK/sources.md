# Sources and version notes

## Primary (current) reference: SC3 Annex A 2025

- Document: **FAI Sporting Code Section 3 – Annex A (Gliding Championships), 2025 edition**
- URL: https://www.fai.org/sites/default/files/sc3a_2025.pdf

Key sections used in this spec:

- **6.1–6.3** — task types and Racing Task definition (incl. recommended ≥45° leg angle).
- **7.4** — starting (Line Start, PEV Start option, Cylinder Start, tolerances).
- **7.6** — Turn Points and Assigned Areas (TP cylinder radius = 500 m for RT).
- **7.8** — finishing (Finish Ring vs Finish Line, ring radius min 3 km, direction for line).
- **7.9** — Task completion definition.

## Historical reference: SC3 Annex A 2013

- Document: **FAI SC3 Annex A 2013_1**
- URL: https://www.fai.org/sites/default/files/documents/sc3a_2013_1.pdf

Why include it?

- Some apps/competitions still use concepts like a **Start Ring** (“leave a circle around the contest site”).
- The 2013 text is explicit about Start Ring geometry, while the 2025 text focuses on Line Start + (waiver-only) Cylinder Start.

Use this file mainly to support *compatibility* modes and importing older task definitions.

## Context: Sporting Code Section 3 (2024)

- Document: **FAI Sporting Code Section 3 – Gliding (2024 edition)**
- URL: https://www.fai.org/sites/default/files/sc3_2024.pdf

Annex A is the “competition rules” layer; Section 3 is broader (records/badges/validation). Most task-creation details are in Annex A.
