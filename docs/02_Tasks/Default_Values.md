# Task Default Values

**Last Updated:** 2026-02-23
**Scope:** Racing and AAT task defaults used by task editors and validation flows.

## Shared Defaults

| Element | Default |
|---|---|
| Start line length | 10 km |
| Start cylinder radius | 10 km |
| Finish line length | 3 km |
| Finish cylinder radius | 3 km |

## Racing Defaults

| Element | Default |
|---|---|
| Racing turnpoint cylinder radius | 0.5 km |
| Keyhole inner cylinder radius | 0.5 km |
| Keyhole outer sector angle | 90 deg |
| Keyhole outer sector radius | 10 km |

## AAT Defaults

| Element | Default |
|---|---|
| AAT circle area radius | 10 km |
| AAT start line length | 10 km |
| AAT finish cylinder radius | 3 km |

## Preservation Rule

User-edited values must be preserved when switching task types unless the user explicitly resets values.
