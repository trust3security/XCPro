# XCPro Agents

This directory contains specialist agent definitions used to guide Codex or other coding agents.

Recommended order of adoption:
1. xcpro-engineer
2. xcpro-test-engineer
3. xcpro-refactor
4. xcpro-performance
5. xcpro-ui-designer

Do not run all agents freely at once.
Start with one agent, one contained task, then tighten the repo contracts based on what goes wrong.
